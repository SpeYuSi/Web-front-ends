### 搭建React开发环境的第一种方法：
 
- 1.全局安装脚手架工具

      `npm install -g create-react-app`
- 2.创建项目

      `create-react-app reactdemo`
- 3.cd  到项目里面 

      `npm start `
       npm run build `
        
### 搭建React的开发环境的第二种方法：

- 1.找到项目要创建的目录执行：

      `npx create-react-app reactdemo` 
- 2.cd  到项目里面 

      `npm start  运行项目（调试）`
      `npm run build 生成项目（发布）`

### npx介绍
-npm v5.2.0引入的一条命令（npx），引入这个命令的目的是为了提升开发者使用包内提供的命令行工具的体验。
